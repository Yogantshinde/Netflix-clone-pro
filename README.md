# Netflix Clone Pro

A Netflix-inspired web application developed using HTML, CSS, and JavaScript.

## Features

* User Login Authentication
* Dashboard Interface
* Movie Search
* Category Filters
* Watchlist Management
* Local Storage Support
* Responsive Design
* Dark/Light Mode

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
